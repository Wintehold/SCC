# Studio Capital — Démonstration premium

Site Front-End Next.js / React / TypeScript construit d’après le cahier des charges Studio Capital.

## Lancement

1. Installer Node.js 20 ou supérieur.
2. Exécuter `npm install`.
3. Exécuter `npm run dev`.
4. Ouvrir `http://localhost:3000`.

## Inclus

- Hero et direction artistique Studio Capital
- Catalogue de 18 logements et filtres instantanés
- Fiches logements en modal
- Simulateur de rentabilité interactif
- Formulaire adaptatif propriétaire / investisseur / voyageur
- FAQ avec recherche
- Témoignages, journal, statistiques et WhatsApp
- Responsive mobile, tablette et desktop
- Métadonnées SEO et composants accessibles
- Données entièrement simulées côté Front-End

Les visuels immobiliers ont été créés spécifiquement pour cette démonstration. Aucune donnée de formulaire n’est stockée.
